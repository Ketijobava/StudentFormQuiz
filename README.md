## რა აქვს აპლიკაციას

- სახელის, გვარის და იმეილის ველები
- თარიღის არჩევა კალენდრიდან
- არჩეული თარიღის ჩვენება `DD/MM/YYYY` ფორმატით
- საყვარელი ჟანრის არჩევა: როკი, კლასიკა ან ჯაზი
- წესებსა და პირობებზე თანხმობის Switch
- Submit ღილაკზე ველების შემოწმება
- Toast შეტყობინებები შეცდომის და წარმატების დროს
